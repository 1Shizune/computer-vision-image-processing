Computer Vision and Image Processing Portfolio

This repository contains eight classical image processing implementations written in C++ and Java.

Topics Include:
- Histogram analysis and thresholding
- Bi-means auto thresholding
- Image enhancement filters (mean, median, Gaussian)
- Morphological operations
- Distance transform compression
- Thinning (skeletonization)
- Connected component labeling
- Chain code boundary representation

Technologies Used:
- C++
- Java
- Image Processing Algorithms
- Binary Image Analysis
