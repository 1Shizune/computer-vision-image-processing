## How to Run

-Make sure all input files (data*.txt and Elem*.txt) are located in the same directory 
as `SanchezP_Project4_Main.java` before compiling and running.

1) javac morphology-operations.java
2) java SanchezP_Project4_Main data1.txt Elem1.txt 5 prettyPrint1.txt
3) java SanchezP_Project4_Main data2.txt Elem2.txt 3 prettyPrint2.txt
4) java SanchezP_Project4_Main data2.txt Elem2.txt 4 prettyPrint3.txt
5) java SanchezP_Project4_Main data3.txt Elem3.txt 4 prettyPrint4.txt
6) java SanchezP_Project4_Main data2.txt Elem3.txt 3 prettyPrint5.txt
